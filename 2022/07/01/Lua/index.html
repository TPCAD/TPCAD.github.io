数据类型

Lua 中有 8 中基本类型。

  • nil
  • boolean
  • number
  • string
  • table
  • function
  • userdata
  • thread

使用 type() 可以输出变量的数据类型。

nil

nil 类型表示没有任何有效值。相当于 C 中的 NULL没有被声明过的变量都是 nil。如果给一个变量赋值 nil，相当如删除这个变量。

boolean

boolean 类型表示真或假，只有两种值：true 或 false。在 Lua 中，只有 false 和 nil 可以表示 false，其他的都为 true，包括数字 0。

Lua 也支持 <、>、<=、>=、==、~= 等关系运算符。但 Lua 的不等于是用 ~= 来表示的。

Lua 的逻辑运算符分别有 and（与）、or（或）、not（非）。Lua 中，逻辑表达式的结果并不完全是 true 或 false。

number

Lua 默认只有一种 number – double 类型。

字符串

Lua 中的字符串用一对双引号或单引号表示。

也可以使用 [[]] 来表示一块字符串。换行符、转义字符等都会被保留。

在 Lua 中可以使用 .. 连接字符串。

当使用 + 连接字符串时，Lua 会尝试将字符串转成一个数字。若字符串无法转成数字，则会报错。

可以使用 # 来获取字符串长度。

tonumber 和 tostring 函数可以让数字和字符串互相转换。若转换失败，结果为 nil。

table

Lua 的 table 有两种下标，一种是数字下标，一种是字符串下标。

数字下标

数字下标的 table，类似于其他语言中的数组。可以存数字、字符串、函数、table 等。

使用大括号即可声明一个 table。

与其他语言不同的是，table 的索引是从 1 开始的。

若 table 中的下标不表示任何内容，则其值为 nil

可以直接对 table 中的元素进行赋值。

可以使用 # 来获取 table 的长度。

insert 函数可以在 table 中插入元素。

remove 函数可以移除 table 中的特定元素。
