Vim

Vi 是一款文本编辑器，几乎所有 Linux 系统都会内置 vi。Vim 是 Vi 的增强版，是目前使用最多的类vi编辑器之一。Vim 通过各种命令与组合键实现文本间的移动和跳转，让你在编辑代码时脱离鼠标，更加高效地进行代码编辑。

本文基于 NeoVim，它是一款基于 Vim 的类vi编辑器。在 99% 支持 Vim 的基础上增加更多新功能。

使用 Vim

安装 NeoVim/Vim 后，在终端输入 vim 即可打开 NeoVim/Vim。后面接上文件名即可打开以 Vim 打开对应文件。

Vim 一般有四个模式：

  • 正常模式（Normal-mode）
  • 插入模式（Insert-mode）
  • 命令模式（Command-mode）
  • 可视模式（Visual-mode）

正常模式

启动 Vim 即会进入正常模式。Vim 的大部分操作都是在正常模式下完成的。正常模式一般用于浏览文件，进行复制、粘贴、删除等操作。在此模式下，按下键盘按键都会被识别为命令，而不是字符。

要进入其他三个模式只能从正常模式进入。在其他三个模式下，按下 esc 即可进入正常模式。

插入模式

在正常模式下按下以下按键即可进入插入模式。

按键 进入位置
i 光标所在处
I 光标所在行第一个非空字符
a 光标所在处下一个字符
A 光标所在行最后一个字符
o 光标所在行的下一行（新的一行）
O 光标所在行的上一行（新的一行）
r 进入取代模式（Replace-mode）。取代光标所在处一次
R 进入取代模式（Replace-mode）。一直取代光标所在处，直至推出

在插入模式下，可以正常输入字符。同时可以使用以下功能按键。

  • Enter：换行
  • Backspace：删除光标前一个字符
  • Delete：删除光标后一个字符
  • 方向键：移动光标
  • Home/End：移动光标至行首或行尾
  • PageUp/PageDown：上下翻页
  • ESC：退出插入模式

命令模式

在正常模式下按下 : 或 / 即可进入命令模式。在命令模式可以执行 Vim 或插件提供的命令。

具体命令见后文。

可视模式

在正常模式下按下 v 即可进入可视模式。在可视模式下，移动光标即可选中文本，选中后可使用正常模式下的命令进行复制、粘贴、删除等操作。

按键说明

正常模式

光标移动

按键 动作
h 光标向左移动一个字符
j 光标向下移动一个字符
k 光标向上移动一个字符
l 光标向右移动一个字符

以上四个按键一次只能移动一个字符，想要实现多次移动，可以先按下数字键，再按下动作。如：30j 表示向下移动 30 个字符。

按键 操作
w 向前移动一个单词
b 向后移动一个单词
gg 移动光标至文档第一行
G 移动光标至文档最后一行
nG n 为数字，移动光标至文档第 n 行
0 移动光标至所在行行首
$ 移动光标至所在行行尾
H 移动光标到屏幕的最上方一行的第一个字符
M 移动光标到屏幕的中央一行的第一个字符
L 移动光标到屏幕的最下方一行的第一个字符
Ctrl+f 向下翻动一页
Ctrl+b 向上翻动一页
Ctrl+d 向下翻动半页
Ctrl+u 向上翻动半页
n<space> n 是数字，向后移动 n 个字符
n<enter> n 是数字，向下移动 n 个字符

复制、粘贴、删除

按键 动作
x 删除光标后一个字符
X 删除光标前一个字符
dd 剪切光标所在行
yy 复制光标所在行
p 粘贴在下一行
P 粘贴在上一行

以上动作均可搭配光标移动操作和数字使用。如：dG 表示删除光标所在行至文档最后一行的内容，d$ 表示删除光标所在行至该行行尾的内容，nyy 表示复制光标所在行向下 n 行的内容。

按键 动作
u 撤销上一动作
Ctrl+r 重做上一动作

命令模式

保存退出

命令 效果
:w 保存文档
:w! 强制保存文档
:q 退出 Vim
:q! 强制退出 Vim（不保存数据）
:wq 保存并退出
:w <filename> 将数据保存到另一个文档
:r <filename> 读入另一个文档的数据
:n1, n2 w <filename> 将 n1 到 n2 行内容保存到另一文档
:! command 在 vim 中执行命令行命令，并显示在 Vim 的底部命令行

搜索替换

搜索
  • /word：向下搜索名为 word 的字符串
  • ?word：向上搜索名为 word 的字符串

确定搜索后可以按 n 移动到下一个搜索到的字符串，N 移动到上一个搜索到字符串。

上一个还是下一个取决于使用的是 / 还是 ?。

若是 ?，则 n 为向上移动，N 为向下移动。

若是 /，则 n 为向下移动，N 为向上移动。

替换

:{作用范围}s/{目标}/{替换}/{替换标志}

作用范围

作用范围有当前行、全文、区间三种。

当前行：

作用范围留空即表示当前行。

:s/node/NODE/g

全文：

使用 % 即表示全文。

:%s/node/NODE/g

区间：

n1,n2s/node/NODE/g

n1，n2 为数字，表示从 n1 行到 n2 行。

在可视模式下选中文本后，按下 : ，Vim会自动补全 :'<,'> 同样表示一个区间。

替换标志

上面的命令末尾都带上了替换标志 g，它表示全局替换。默认只会替换一行中第一次出现的对应字符串。

  • i：大小写不敏感
  • I：大小写敏感
  • c：替换前确认

替换标志可以组合使用。如：

:s/node/NODE/gI
