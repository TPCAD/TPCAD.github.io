Hexo Note

安装 Hexo 前需先安装 Git 和 Node.js

安装 Node.js

Ubuntu 可以使用sudo apt install nodejs安装 Node.js，但安装的 Node.js 版本为 10.19.0

该版本使用 nodejs作为命令开头

使用如下方式安装最新版

将 16 换成其他数字即可安装对应版本

安装完成后使用 node -v 查看 Node.js 版本

安装 Hexo

建站

搭建博客前务必配置 Git

初始化

执行如下命令，在当前目录下初始化一个博客

初始化完成后可以使用 hexo s 或 hexo server 运行 Hexo

执行命令后可在 https://localhost:4000 访问到 Hexo 博客

但这只是本地运行的博客，只能在自己的电脑上访问

Ctrl + c 可以关闭 Hexo

执行 hexo deploy 后会提示输入 Github 用户名及密码。若报错可尝试重试或配置 Git

  1. 提示部署成功后刷新 Github 可以看到仓库已更新。通过 https://username.github.io 访问 Hexo 博客

基础配置

在 _config.yml 中修改对应选项，为博客添加基本信息

更换博客主题

  1. 在 Hexo官网 找到喜欢的主题的 Github 仓库，复制 https 链接
  2. 使用 Git 命令将主题克隆到 themes 文件夹（themes 文件夹位于博客所在目录下）

从 Github 下载文件过程非常慢

  1. 配置 _config.yml

在 _config.yml 中找到 theme，将原有的 landscape 改为主题的文件夹名，如刚才的 next

  1. 更新 Hexo 博客

主题并不会立刻生效，多刷新几次

配置 Next 主题

NexT 主题的配置文件位于 hexo-site/themes/next/_config.yml

切换方案（主题）

NexT 内置了 Muse、Mist、Pisces、Gemini 四种方案（Scheme）

可在 NexT 的配置文件中切换主题

网页图标

将网页图标放在 hexo-site/sources/images/ 目录下。

修改配置文件中的相关选项。支持适配不同设备

自定义 LOGO

LOGO 显示在侧边栏

与网页图标相同，将图片放在 hexo-site/sources/images/ 下并修改配置文件即可

菜单

除了 home 和 archives ，其他的选项都需要先创建 page

侧边栏

头像

将图片放在 hexo-site/source/images/ 目录下，并修改上方的 url 项

rounded 可以将头像放在一个圆框内

rotated 可以使头像在鼠标悬停时旋转

社交链接

社交链接可以在侧边栏显示 Github、E-Mail、Weibo、Google、Twitter 等链接

修改对应选项即可。以 Github 为例，取消该行注释，并修改链接

|| 前面是永久链接，后面是 fontawesome 的图标

默认会同时显示图标和文字，如果想只显示图标，可以把 icons_only 设为 true

除了以上这些选项，也可以自定义添加，比如微信、QQ等，但图标必须是来自 fontawesome
