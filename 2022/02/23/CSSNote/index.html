CSS Note

在 HTML 中应用 CSS

外部样式表

在 <head> 中通过 <link> 标签链接一个 .css 文件

内部样式表

不使用外部文件，将 CSS 放在 <head> 标签里的 <style> 标签中

内联样式表

在某一个 html 标签中使用 style 属性

一些概念

选择器

选择器用于确定样式表该作用于 HTML 文档的哪一部分

选择器有很多，常见的有 元素选择器，如：h1、p。还有 类选择器、ID选择器、伪类选择器

空白

CSS 和 HTML 一样，会忽视文档中的大部分空白，但不是所有，如属性的值与值之间的空白不会被忽视，属性名称中空白不会被忽视

选择器

类型选择器

类型选择器也叫“标签选择器”或“元素选择器”，因为它使用的是 HTML标签

全局选择器

全局选择器可以选中文档中的所有内容。也可以跟在其他选择器后面，表示选中该选择器的子内容

<ul> 的 border 属性原本不会被其子元素继承，但添加全局选择器后，子元素也被选中了

类选择器

class 属性

在一个 class 属性中可以指定多个值，用空格进行分隔

一个 class 属性可以被添加给多个相同或不同的元素

类选择器以句点开头 .，使用前需要先给 HTML 标签添加 class 属性

如果有多个不同类型的标签拥有同一个 class 属性，但指向选中某一种标签

当一个元素拥有多个 class 时，只要选择器出现其中一个 class，该元素都可以被选中

ID选择器

id 属性

在 HTML 文档中一个 id 只允许出现一次

ID选择器以 # 开头，用法与类选择器相似
