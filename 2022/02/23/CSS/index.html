CSS 和 HTML 一样，会忽视文档中的大部分空白，但不是所有，如属性的值与值之间的空白不会被忽视，属性名称中空白不会被忽视

选择器

类型选择器

类型选择器也叫“标签选择器”或“元素选择器”，因为它使用的是 HTML 标签

全局选择器

全局选择器可以选中文档中的所有内容。也可以跟在其他选择器后面，表示选中该选择器的子内容

<ul> 的 border 属性原本不会被其子元素继承，但添加全局选择器后，子元素也被选中了

类选择器

class 属性

在一个 class 属性中可以指定多个值，用空格进行分隔

一个 class 属性可以被添加给多个相同或不同的元素

类选择器以句点开头 .，使用前需要先给 HTML 标签添加 class 属性

如果有多个不同类型的标签拥有同一个 class 属性，但只想选中某一种标签

当一个元素拥有多个 class 时，只要选择器出现其中一个 class，该元素都可以被选中

ID 选择器

id 属性

在一个 HTML 文档中一个 id 只允许出现一次

ID 选择器以 # 开头，用法与类选择器相似

伪类选择器

伪类选择器用于选择处于特定状态打元素。比如某种类型的第一个元素，当鼠标指针悬停在元素上面打时候。效果看起来就像应用了类一样，它可以有效减少多余的类。

伪类选择器以 : 开头。

详细的伪类用法可以查看 MDN 及 W3C。

常用的伪类选择器：

:active

在用户激活（如点击）元素时匹配。

当鼠标点击文本时，背景会变为粉色。

:active 常用于 a 标签。与它类似的选择器还有 :link、:hover、:visited。使用这四个伪类选择器时应该遵循 LVHA 顺序。

  • :link 链接访问前
  • :visited 链接访问后
  • :hover 鼠标悬停时
  • :active 鼠标点击后
:first-child

选择一组兄弟元素中的第一个元素。类似打有 :last-child。

:first-of-type

选择一组兄弟元素中其类型的第一个元素。

last-of-type 可以选择一组兄弟元素中其类型的最后一个元素。

:focus

表示获得焦点的元素（如表单输入）。当用户点击或触摸元素或通过键盘的 “tab” 键选择它时会被触发。

focus-within 可以表示元素或该元素的后代元素获得焦点。

盒模型

元素的对外显示类型

元素对外显示类型主要有三种，块（block）、内联（inline）、内联块（inline-block）。

块级盒子会表现出以下行为：

  • 每个盒子都会换行
  • width 和 height 属性可以生效
  • margin、padding、boder 会将其他元素从当前盒子“推开”

内联盒子会表现出以下行为：

  • 盒子不会产生换行
  • width 和 height 属性不会生效
  • 垂直方向的内边距、外边距以及边框会被应用但是不会把其他处于 inline 状态的盒子推开。
  • 水平方向的内边距、外边距以及边框会被应用且会把其他处于 inline 状态的盒子推开。

内联块是介于块和内联之间打一种状态，它不会产生换行，但 width 和 height 可以生效，且 margin、padding、boder 会将其他元素从当前盒子“推开”

使用 display 属性可以改变元素的显示类型。

盒模型的组成

  • content：显示内容，大小通过 width 和 length 设置
  • padding：内边距，内容到边框的距离，大小通过 padding 设置
  • border：边框，大小通过 border 设置
  • margin：外边框，盒子与其他元素之间的距离，大小通过 margin 设置

标准盒模型的大小由 content、padding、border 共同决定。如，一个 width 为 300 的盒子，加上大小为 10 的 border 后，盒子实际大小为 320。所以边框和内边距会影响盒子的实际大小。

背景

background 属性可以为元素添加背景。

背景颜色

background-color 用于设置背景颜色。该属性接受任意有效打 <color> 值。

<color> 值有以下定义方式：

  • 关键字，如 blue、black
  • RGB 立体坐标（十六进制表示或 rgb()、rgba() 函数表示）
  • HSL 圆柱坐标（hsl() 、 hsla() 函数表示）

background-color 默认值为 transparent，即透明。

不透明度

使用 rgba() 表示颜色时，需要四个参数，前三个表示 RGB 数值，最后一位表示不透明度，取值在 0~1 之间，数值越大不透明度越大。

背景图片

background-image 可以以图片作为背景。该属性接受一个 url()。

背景图片相比于 <img>，更容易进行定位。

如果同时设置了背景颜色和背景图片，则图片会显示在颜色上方。

background-image 的默认值为 none，即无图片。

背景图片平铺

background-repeat 属性用于控制背景图片的平铺行为。

它有以下几个值：

  • no-repeat：不平铺
  • repeat-x：水平平铺
  • repeat-y：垂直平铺
  • repeat：水平与垂直平铺

background-repeat 的默认值为 repeat。

背景图片位置

background-position 可以设置背景图片的位置。background-position 接受两个值来确定位置，x 与 y。

值可以是方位关键字：top、bottom、left、right、center。

x 控制左中右（left、center、right），y 控制上中下（top、center、bottom）。

背景附着

background-attachment 可以设置背景图片是否跟随对象滚动。

该属性有两个值：

  • scroll：滚动，默认值
  • fixed：固定

内边距

padding 可以设置盒子的内边距。

padding属性的简写可以接受 1~4 个值。

  • 1 个值，同时设置上下左右内边距
  • 2 个值，第一个代表上下内边距，第二个代表左右内边距
  • 3 个值，第一个代表上内边距，第二个代表左右内边距，第三个代表下内边距
  • 4 个值，分别代表上、右、下、左，顺时针方向。

因为内边距是盒子实际大小的一部分，所以设置内边距会将原来的盒子撑大，但只有在设置了 width 或 height 之后，对应的内边距才会撑大盒子。

外边距

margin 设置盒子的外边距。

margin 的外边距简写与 padding 相同。

外边距折叠

使用 margin 定义块元素的垂直外边距时，可能会出现外边距折叠的情况。

相邻块元素垂直外边距折叠

解决方法是只设置一个盒子的外边距。

嵌套块元素垂直外边距折叠

阴影

盒子阴影

box-shadow 可以为盒子添加阴影。

  • h-shadow：必选，水平阴影位置，数值越大，阴影向右偏移，允许负值
  • v-shadow：必选，垂直阴影位置，数值越大，阴影向下偏移，允许负值
  • blur：可选，模糊距离
  • spread：可选，阴影尺寸
  • color：可选，阴影颜色
  • inset：可选，内部阴影。默认为 outset

默认 outset 不需要声明，若声明，阴影会消失。

阴影不占盒子空间，不影响盒子排列。

文字阴影

text-shadow 可以为文字添加阴影。

浮动

浮动可以实现多个块级元素横向排列。

float 可以为元素添加浮动效果。它有三个可选值。

  • none：无浮动。默认
  • left：左浮动
  • right：右浮动

浮动特性

脱标

脱标即脱离标准流。

将网页想象成房子中的一层楼，元素在这层楼中占据着自己的位置。浮动会让元素浮起来，进入第二层楼，处于第二层楼的元素将会遮蔽第一层楼的元素。浮动的元素不会保留原来的位置，而是移动到其直接父元素的边框或同行浮动元素的边缘。多个浮动的盒子之间会紧密贴在一起，没有缝隙。

一行显示

如果多个盒子都设置了浮动，则这多个盒子会在同一行显示（必须处于同一直接父元素）。紧密相连，顶端对齐
